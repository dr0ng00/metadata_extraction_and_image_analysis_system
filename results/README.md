# Results Directory

This directory contains all outputs generated by the MetaForensicAI system. All analysis reports and data exports are stored here.

## Directory Structure

### 📁 reports/
Contains forensic analysis reports in multiple formats:
- **pdf/**: Professional PDF reports
- **json/**: Structured JSON data
- **html/**: Interactive HTML reports

### 📁 exports/
Contains exported data for integration:
- **csv/**: Tabular data for spreadsheets
- **xml/**: Structured data for systems
